O que é a Web?

A World Wide Web (WWW), também conhecida como a web, é um sistema de documentos interligados acessados via internet. Esses documentos são criados usando a linguagem HTML e podem incluir links para outros documentos e recursos.

Os documentos web são armazenados em servidores e acessados por meio de navegadores. Esses navegadores interpretam e exibem os documentos HTML, permitindo que os usuários visualizem e interajam com o conteúdo.

O desenvolvimento da web começou na década de 1990 e desde então tem evoluído para incluir uma ampla gama de tecnologias e padrões, incluindo HTML5, CSS e JavaScript.

Uma página web básica pode incluir elementos como títulos, parágrafos, imagens e links, todos definidos por tags HTML. Aqui está um exemplo básico de uma estrutura de página web:

<html>
    <head>
        <title>Título da Página</title>
    </head>
    <body>
        <h1>Título Principal</h1>
        <p>Este é um parágrafo de exemplo.</p>
        <img src="imagem.jpg" alt="Descrição da Imagem">
    </body>
</html>

Desenvolvido como um exemplo simples de HTML.
