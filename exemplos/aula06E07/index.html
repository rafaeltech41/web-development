CSS

CSS (Cascading Style Sheets – Folha de estilos) é usado para estilizar a página WEB

Três maneira de implantar o CSS

CSS Inline: é adicionado à tag HTML. Excelente para aplicar estilos únicos. Usa o atributo style dentro da tag

Css interno

CSS Interno: é adicionado no <head/> da página WEB. É ideal para uma landing page. Utiliza a tag <style/>

CSS Externo:

CSS Externo: é criado um documento de estilo.css e adicionado o caminho na página WEB. Ideal para várias páginas com o mesmo estilo
