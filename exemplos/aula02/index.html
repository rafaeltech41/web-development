Hypertext Markup Language.

Linguagem de Marcação de textos para criação de página(s) da internet

Desenvolvido por Tim Berners-Lee na década de 1990, baseada na Linguagem SGML

Mantido Oficialmente pelo W3C (World Wide Web Consortium)

Composto por <tags> para adicionar elementos de conteúdos na página WEB


<title>Meu Site</title>
<p>Estamos aprendendo HTML</p>
<img src=“imagem.png”>
